To deploy application on Heroku follow these steps:

! Before anything, upload project code to GitHub

! Make sure url is set up for DATABASE_URL


heroku login
heroku create (app name)
heroku addons:create heroku-postgresql --app (app name)
heroku pg:promote DATABASE --app (app name)
git push heroku master
heroku open
heroku logs --tail


####################################################


To deploy application on AWS follow these steps:

! Set up your DB first!!!! Make sure you have access to security group

! Change app variable to application (use refactoring)

! Add .ebextensions folder containing application.config

! Make sure to check IAM on AWS for access-key


eb init
eb create
eb open
eb logs
