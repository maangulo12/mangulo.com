AWS Amazon Web Services

AWS is a collection of cloud computing services (web services) that anyone can use for FREE for 1 year.

These are some of the services that a simple web app would use in AWS:
  • EC2 - Cloud compute instances (virtual servers)
  • *S3 - Cloud storage (usually static files)
  • *RDS - Relation database service in the Cloud (PostgreSQL, MySQL, Oracle, etc..)
  • *Elastic Beanstalk - Easy to use service for deploying and scaling your web apps in the Cloud
    (Download EB CLI Elastic Beanstalk Command Line Interface)

Heroku Cloud Platform as a Service

Heroku is a cloud platform built on top of AWS that allows users to easily deploy web apps to the cloud for FREE!

Heroku is great for testing and deploying your web app to the World Wide Web.
Follow this guide to deploy a Python web app to Heroku.
I recommend to download the Heroku Toolbelt

GitHub Open Source Project Hosting

GitHub is an open source project hosting service that allows users to share, collaborate, and keep track of their source code projects.

I recommend everyone to create an account on GitHub to showcase your projects.
